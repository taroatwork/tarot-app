# タロット・リーディング（大アルカナ22枚）

ブラウザだけで完結するタロット占いアプリ。ビルド不要の静的サイト（`index.html` 1枚）です。

## 特徴
- 大アルカナ22枚から、必要枚数を自分で選ぶ
- スプレッド3種：ワンオラクル / スリーカード / ケルト十字
- ケルト十字は「左の十字＋右の杖」の正式配置で描画
- 正位置/逆位置で絵柄が反転、吉凶を色分け表示
- 質問への「結論」と「次の一歩」をセクションで解説
- 結果はブラウザの localStorage に保存（履歴閲覧）。サーバー送信なし

## ローカルで開く
`index.html` をブラウザで開くだけで動きます。

## デプロイ（Vercel）
静的サイトなので設定不要。GitHub にプッシュして Vercel でインポートすればそのまま公開できます。

## 技術
- HTML / CSS / Vanilla JavaScript（フレームワーク・ビルド工程なし）
- アイコン：Tabler Icons（CDN）/ フォント：Noto Sans JP・Noto Serif JP（Google Fonts）

本アプリはエンターテインメント目的です。
